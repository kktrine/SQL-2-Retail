## Chapter I

## Introduction

В данном проекте реализована модель сети магазинов розничной торговли. У всех магазинов единая база транзакций, чеков, товаров и клиентов. Товары разделены по категориям. Клиенты имеют карты магазина, номера которых запоминаются при покупке.

## Part 4. Формирование персональных предложений, ориентированных на рост среднего чека

Скрипт *part4.sql*, содержащий функцию (*функция в процессе написания, название будет добавлено позже*).

Параметры функции:
- метод расчета среднего чека (1 - за период, 2 - за количество)
- первая и последняя даты периода (для 1 метода)
- количество транзакций (для 2 метода)
- коэффициент увеличения среднего чека
- максимальный индекс оттока
- максимальная доля транзакций со скидкой (в процентах)
- допустимая доля маржи (в процентах)

На основе данных о покупках клиента, индексе востребованности и оттока для каждого клиента формируется персональное предложение скидки на товары определенной группы. Максимальный процент скидки также высчитывается программно.

## Part 5. Формирование персональных предложений, ориентированных на рост частоты визитов

Скрипт *part5.sql*, в котором реализована функция *fnc_increase_visits_frequency*.

Параметры функции:
- первая и последняя даты периода
- добавляемое число транзакций
- максимальный индекс оттока
